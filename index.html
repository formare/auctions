Auction Theory Toolbox

The Auction Theory Toolbox aims at enabling auction designers to verify their own auction designs using mechanised reasoning tools. It provides an infrastructure of basic auction formalisations to build on.
While this new homepage is being created, please see the old homepage and our source repository for full information.
